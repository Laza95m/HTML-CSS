git init
Создает репозиторий (локально)

git status
Позволяет получить текущий статус файлов
кто изменен, кто неотслеживается и т д

git add <filename> || .
Добавляет в отслеживаемые файлы. После этой команды
все изменения в файле будут отслежены.
Эту команду надо делать всегда перед тем, как 
сделать commit

git commit -m 'Сообщение'
Создает "контрольную точку", так называемый
коммит. Регестрирует и запоминает ВСЕ изменения, которые
случились в ОТСЛЕЖИВАЕМЫХ файлах с прошлого коммита

git remote 
Подключение к какому-то репозиторию


git clone <url> 
Клонирование (копирование) всего, что есть
в каком-то репозитории к вам на компьютер

git push -u origin <branch_name> 
Сохранение последнего коммита в Гитхабоский или 
Гитлабовский репозиторий. То есть изменение файлов в 
репозитории
